# Ndustrious — CSS Template

A clean and professional **CSS website template** designed for industrial or business use cases. Minimal, elegant, and fully responsive.

## ✨ Features

- 🎨 Clean industrial-inspired design
- 📱 Fully responsive layout
- ⚡ Lightweight — pure CSS with no heavy frameworks
- 🔧 Easy to extend and customize
- 🌐 Cross-browser compatible

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/hamdyelbatal122/ndustrious.git
   cd ndustrious
   ```

2. **Open in browser**
   ```bash
   open index.html
   ```

## 📁 Project Structure

```
ndustrious/
├── index.html      # Main HTML file
├── style.css       # Core stylesheet
├── images/         # Image assets
└── README.md
```

## 🎨 Customization

Modify the CSS variables to match your brand:

```css
:root {
  --primary-color: #2C3E50;
  --accent-color:  #E74C3C;
  --font-main:     'Roboto', sans-serif;
}
```
